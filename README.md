CrowdSense AI – Smart Crowd Monitoring System

CrowdSenseAI  is a standalone PyQt5 desktop application that leverages YOLOv8 deep learning for real-time crowd monitoring. Designed for speed, reliability, and accessibility, it transforms traditional reactive CCTV surveillance into predictive crowd safety.

Key Features
	•	Real-time detection and counting of people from live feeds, videos, or images using YOLOv8
	•	Instant visual and audible alerts when density thresholds are breached
	•	Standalone, local-first processing with no cloud or internet dependency
	•	Runs on standard laptops/PCs and works with existing CCTV or IP cameras
	•	Easily scalable from single-camera setups to multi-camera networks

Tech Stack
	•	Python – Core development language
	•	PyQt5 – Interactive desktop GUI
	•	OpenCV – Video feed handling and pre-processing
	•	YOLOv8n – Lightweight, high-speed detection model
	•	NWPU-Crowd Dataset – Used for robust training and fine-tuning

Innovation
	•	Operates reliably in low-connectivity or rural environments
	•	Requires no new infrastructure, only existing camera systems
	•	Non-intrusive design that avoids personal identification
	•	Optimized for speed through frame skipping and GPU acceleration, with accuracy boosted via OpenCV enhancements

Impact
	•	Public Safety: Helps prevent stampedes by delivering proactive crowd density alerts
	•	Economic: Reduces reliance on manual monitoring and lowers security overhead costs
	•	Scalable Vision: Future-ready for Smart Cities, transportation hubs, malls, and mass gatherings
